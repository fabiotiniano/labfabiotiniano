7° Clase - Set 20, 2023

Todo esta dentro de una caja

En esta segunda clase presencial aprendimos que cada caja engloba un elemento o agrupación. Etiquetas adicionales como link, style y atriutos como href, type, rel, id y class. Formatos para introducir colores: valores rgb, codigo hex y nombres de color. La opacidad que nos permite sobreponer colores si es que fuera necesario o de nuestro gusto. Colores HSL (HUE: tono, Saturacion: desde 0-100%, Lightness: brillo) CSS: Comandos y reglas adicionales. Además hicimos ejercicios jugando con las escalas de colors y saturación.
